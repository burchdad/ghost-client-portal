# Logo Rebrand Onboarding

The onboarding route is `/projects/[projectId]/onboarding`.

It supports draft saving, required-field validation, completion percentage, file metadata capture, file type and size validation, final submission, client activity, audit and outbox events, related action completion, and Brand Discovery phase completion.

Supported initial file types: SVG, PNG, JPG, JPEG, PDF, AI, EPS, ZIP. SVG files are accepted as uploads but must not be rendered inline without sanitization.
